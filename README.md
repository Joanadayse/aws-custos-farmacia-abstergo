## Introdução
Este relatorio apresenta o processo de implementação de ferramentas na empresa Abstergo Industries , realizado por Joana Dayse. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especi­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon S3 (Simples Storage Service)
- Armazenamento de arquivos grandes e backups.
- Os videos de treinamentos , relatorios médicos digitalizados e notas fiscais eletronicas passaram a ser armazenados no S3 Intelligents-Tiering , reduzind custos de armazenamento sem comprometer a disponibilidade.

Etapa 2: 
- Amazon RDS (Relational Database Service)
- Gerenciamento de banco de dados relacional.
- O sistema de controle de estoque e vendas da farmácia foi migrado para o RDS MySQL, eliminando custos de manutenção de serviços locais e garantindo alta disponibilidade com backups automaticos.

Etapa 3: 
- Amazon CloudFront
- Distribuição de conteúdo com baixa latência.
- O site da farmacia e o portal de clientes passaram a usar CloudFront para entregar páginas de catálogos de produtos com mais rapidez. Isso reduziu custos de largura de banda e melhorou a experiência dos usuários.
